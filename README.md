# mastersheet

A tool for joining and pivoting many spreadsheets.

### To use
Download `mastersheet.html` and open it with a browser.

### Notes
It's a single HTML file with no network access, backend, or storage. \
Feel free to request changes via a GitHub Issue or Discussion.
